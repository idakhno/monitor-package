Проект для тестового задания по реализации скрипта мониторинга.

Лог работы мониторинга сохраняется в /var/log/app_monitor.log.

Состав:
hello.py - HTTP‑сервер, который отдаёт строку «Hello World».
monitor.sh - проверяет доступность приложения, пишет краткий лог и при необходимости перезапускает его.
monitor.conf - настройки, которые использует скрипт мониторинга.
app-monitor.service и app-monitor.timer - файлы systemd для автоматического запуска мониторинга каждые 10 секунд.
install.sh - установка всех файлов и включение таймера.



Как запустить:
1. sudo bash install.sh
2. curl http://127.0.0.1:8080, выводится строка Hello World.
3. tail -f /var/log/app_monitor.log → строки с OK.
4. pkill -f hello.py → в логе появится DOWN и приложение перезапустится.
